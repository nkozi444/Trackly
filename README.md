# 💰 Trackly

Trackly is a full-stack personal finance tracker designed to help users manage their money with confidence. It allows users to record income and expenses, create category-based budgets, monitor savings goals, and gain insights into their spending habits through interactive dashboards and visual analytics.

The application aims to provide a simple, intuitive, and responsive experience for tracking daily finances while promoting better budgeting and financial planning.

## ✨ Features

* 🔐 Secure user authentication (Sign Up & Login)
* 💸 Income and expense transaction management
* 📂 Category-based budgeting
* 📊 Interactive financial dashboard with charts
* 🎯 Savings goal tracking with progress monitoring
* 📈 Budget progress indicators
* 📱 Responsive design for desktop and mobile devices
* 📤 CSV export *(optional)*
* 🔄 Recurring transactions *(optional)*

## 🛠️ Tech Stack

### Frontend

* React
* Vite
* React Router
* CSS / Tailwind CSS (depending on implementation)
* Chart.js or Recharts

### Backend

* Node.js
* Express.js
* JWT Authentication
* bcrypt

### Database

* PostgreSQL
* Prisma ORM (or raw SQL)

### Deployment

* Frontend: Vercel
* Backend: Railway
* Database: Supabase PostgreSQL

## 🎯 Project Goal

Trackly was built to provide a practical and user-friendly budgeting solution while demonstrating full-stack web development concepts, including authentication, RESTful APIs, database design, CRUD operations, and data visualization.
